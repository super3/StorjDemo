# Farming Demo on Storj Platform

## Requirements
This requires 
* `g++` (v4.8 or higher for test)
* `python` (2.7.x ,3.3.x, or 3.4.x)

## Installation

To install it,

    $ python setup.py install

To run the associated tests.

    $ PYTHONPATH=. py.test -q tests/test.py -s
